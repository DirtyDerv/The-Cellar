# Components Directory

This directory contains React components for The Cellar app.

## Structure

- `ui/` - Shadcn UI components (Button, Card, Input, etc.)
- `camera/` - Camera capture and scanning components
- `scans/` - Scan results and history components
- `dashboard/` - Manager dashboard components

## Adding Shadcn UI Components

To add a new Shadcn UI component:

```bash
npx shadcn@latest add button
npx shadcn@latest add card
npx shadcn@latest add input
# etc.
```

Components will be added to the `ui/` subdirectory automatically.
